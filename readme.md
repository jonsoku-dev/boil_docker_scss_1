## git clone

```bash
$ git clone https://github.com/rakutenjongseok/docker-sass.git
```

## 폴더 이동

```bash
$ cd docker-sass
```

## docker-compose

```bash
$ docker-compose up
```

## 컨테이너 안에 들어가보기

```bash
$ docker ps -a
$ docker exec -it [container Id] /bin/sh
or
$ docker exec -it [container Id] /bin/bash
```
